# andrewvos.com

Personal site served by Bun's built-in HTTP server with React server rendering.

## Development

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Commands

```bash
bun run check
bun run build
bun run test
bun run start
```

`bun run test` typechecks, builds the Tailwind and browser assets, then smoke-tests the Bun server routes.
