# andrewvos.com

This project now runs as a Bun server using `Bun.serve` (no Next.js runtime).

## Getting Started

```bash
bun install
bun run dev
```

Open `http://localhost:3000`.

## Scripts

- `bun run dev` - run the Bun server with hot reload
- `bun run start` - run the Bun server
- `bun run lint` - lint `server.ts` and scripts
- `bun run scrape-book` - run book scraper
- `bun run renew-book-thumbnails` - refresh book thumbnails
